# Decisiones

Para nuevos Registros de Decisiones Arquitectónicas (ADRs), use una de las siguientes plantillas como punto de partida:

* [adr-template.md](adr-template.md) contiene todas las secciones, con explicaciones sobre ellas.
* [adr-template-minimal.md](adr-template-minimal.md) solo contiene las secciones obligatorias, con explicaciones sobre ellas. <!-- ### Consecuencias también incluida, aunque marcada como "opcional" -->
* [adr-template-bare.md](adr-template-bare.md) contiene todas las secciones vacías (sin explicaciones).
* [adr-template-bare-minimal.md](adr-template-bare-minimal.md) contiene las secciones obligatorias, sin explicaciones. <!-- ### Consecuencias también incluida, aunque marcada como "opcional" -->

La documentación de MADR está disponible en <https://adr.github.io/madr/> y la información general sobre ADRs está disponible en <https://adr.github.io/>.
